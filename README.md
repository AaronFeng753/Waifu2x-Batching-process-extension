# Waifu2x-Batching-process-extension
Waifu2x-Batching process extension for waifu2x-ncnn-vulkan

This extension is made for waifu2x-ncnn-vulkan, with this extension, you can use waifu2x to process all the images in one folder, without input all file one by one.

waifu2x-ncnn-vulkan:
https://github.com/nihui/waifu2x-ncnn-vulkan

# Waifu2x批量处理扩展
适用于waifu2x-ncnn-vulkan的批量图片处理扩展

这个扩展适用于waifu2x-ncnn-vulkan, 借助这个扩展, 你可以使用waifu2x-ncnn-vulkan来一次性放大一个文件夹内的所有图片, 而不用一个个输入文件.

waifu2x-ncnn-vulkan:
https://github.com/nihui/waifu2x-ncnn-vulkan
